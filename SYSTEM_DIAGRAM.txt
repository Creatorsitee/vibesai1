═══════════════════════════════════════════════════════════════════════════════
REQUEST/RESPONSE CYCLE (EXAMPLE: CHAT MESSAGE)
═══════════════════════════════════════════════════════════════════════════════

User Types Message
        │
        ▼
ChatPanel Component
        │
        ▼
onSendMessage Callback
        │
        ├─→ Validation: Check API key initialized
        │
        ├─→ Update State: addMessage('user', text)
        │
        ├─→ Show Loading: setIsProcessing(true)
        │
        ▼
GeminiService.generateContentStream()
        │
        ├─→ Retry Logic: with exponential backoff
        │
        ├─→ Timeout: 30 seconds max
        │
        ├─→ Fetch: Send to Gemini API
        │
        ├─→ Stream: Receive chunks
        │
        ▼
Chunk Handler: onChunk()
        │
        ├─→ Update State: updateLastMessage(fullText)
        │
        ▼
ChatPanel Re-renders
        │
        ├─→ Display: New message appears
        │
        ├─→ Scroll: Auto-scroll to bottom
        │
        ▼
User Sees Response
